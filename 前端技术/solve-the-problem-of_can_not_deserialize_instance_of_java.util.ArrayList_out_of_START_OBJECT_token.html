# 问题解决

主要问题出在前端请求参数有问题，当参数为List时直接传数组，而@QueryParam的参数需要带在Url参数后面。

前端请求(简写)：

url: http://test.com/finishOrder?hasSix=N
headers:token:123456
method:post
request Body:
[{"num":"202103192374","CustomerIndex":0,"RemarkIndex":"","RemarkName":"","RemarkValue":"","Rid":"0"}]

此时，问题解决。
